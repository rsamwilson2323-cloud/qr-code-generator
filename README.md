# 📱 QR Code Generator

A simple and modern QR Code Generator built using HTML, CSS, and JavaScript. This web app allows users to generate QR codes instantly for any text or URL and download them with a single click.

---

## 🚀 Features

- ⚡ Instant QR code generation  
- 🌐 Supports text and URLs  
- 🎨 Clean and modern UI design  
- ⬇️ One-click QR code download  
- 🔗 Uses online QR API (no heavy libraries)  
- 💻 Lightweight and fast  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript  
- QR Code API

## ⚙️ How to Run

### 1️⃣ Clone the repository
git clone https://github.com/rsamwilson2323-cloud/qr-code-generator.git

### 2️⃣ Navigate into the folder
cd qr-code-generator

### 3️⃣ Open the project
Double click index.html  
OR run:

start index.html      (Windows)
open index.html       (Mac)
xdg-open index.html   (Linux)

---

## 🎯 Usage

1. Enter any text or URL  
2. Click "Generate QR"  
3. QR code appears instantly  
4. Click "Download QR" to save
